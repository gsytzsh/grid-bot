# OKX 网格交易机器人

一个基于 Python 的 OKX 现货网格交易系统，带 Web 管理面板。

## 功能特性

- ✅ **网格交易策略**: 自动低买高卖，震荡行情神器
- ✅ **止盈止损**: 全局止损/止盈，保护本金
- ✅ **Web 面板**: 实时查看网格状态、利润、创建新网格
- ✅ **日志记录**: 所有交易操作留痕
- ✅ **自动运行**: 启动后无需盯盘，自动执行买卖

## 快速开始

### 环境要求

- Python 3.8+
- macOS / Linux / Windows
- OKX 账户（已实名认证）

### 1. 安装依赖

```bash
cd okx-arb-bot
pip3 install -r requirements.txt
```

如果提示权限错误，使用虚拟环境：

```bash
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 2. 获取 OKX API 密钥

1. 登录 OKX 官网或 App
2. 进入 **设置** → **API 管理**
3. 点击 **创建 API**
4. 填写信息：
   - API 名称：随意（如 `grid-bot`）
   - 权限：勾选 **读取** + **交易**（不要勾选提现）
   - 密码验证：输入你的资金密码
5. 保存以下信息（只显示一次！）：
   - `API Key`
   - `Secret Key`
   - `Passphrase`

### 3. 配置 API

```bash
cp config/.env.example config/.env
```

编辑 `config/.env` 文件：

```bash
# OKX API 配置（必填）
OKX_API_KEY=你的_api_key
OKX_SECRET_KEY=你的_secret_key
OKX_PASSPHRASE=你的_passphrase

# 交易配置（可选）
STOP_LOSS_PERCENT=5      # 全局止损 5%（跌破此比例停止网格）
TAKE_PROFIT_PERCENT=20   # 全局止盈 20%（涨破此比例停止网格）

# 服务器配置（可选）
WEB_HOST=127.0.0.1
WEB_PORT=8000
```

### 4. 启动机器人

```bash
# 如果使用虚拟环境
source venv/bin/activate

# 启动
python main.py
```

看到以下输出表示成功：

```
INFO - 正在启动 OKX 网格交易机器人...
INFO - ✓ Bot 初始化完成
INFO - Web 面板地址：http://127.0.0.1:8000
```

### 5. 访问 Web 面板

打开浏览器访问：**http://127.0.0.1:8000**

## 使用指南

### 创建第一个网格

1. 点击页面右上角 **「+ 创建网格」**
2. 填写参数：

| 字段 | 说明 | 示例 |
|------|------|------|
| 交易对 | 选择要交易的币种 | BTC-USDT |
| 价格下限 | 网格最低价格 | 85000 |
| 价格上限 | 网格最高价格 | 115000 |
| 网格数量 | 分成多少格 | 10 |
| 投资金额 | 投入多少 USDT | 100 |
| 止损价 | 可选，跌破此价停止 | 80000 |
| 止盈价 | 可选，涨破此价停止 | 120000 |

3. 点击 **「创建网格」**
4. 点击 **「启动」** 开始自动交易

### 推荐参数（100 USDT 测试）

假设 BTC 当前价格 **$100,000**：

```
交易对：BTC-USDT
价格下限：85,000  （当前价 -15%）
价格上限：115,000 （当前价 +15%）
网格数量：10 格
投资金额：100 USDT
止损价：80,000    （可选）
止盈价：120,000   （可选）
```

### 网格参数计算

```
每格间距 = (上限 - 下限) / 网格数
         = (115000 - 85000) / 10
         = 3,000 (约 3%)

每格金额 = 投资金额 / 网格数
         = 100 / 10
         = 10 USDT

每格利润 = 每格金额 × 间距 - 手续费
         = 10 × 3% - 0.016
         ≈ 0.28 USDT
```

### 快速计算表

| BTC 价格 | 下限 (-15%) | 上限 (+15%) |
|---------|------------|------------|
| 50,000  | 42,500     | 57,500     |
| 80,000  | 68,000     | 92,000     |
| 100,000 | 85,000     | 115,000    |
| 150,000 | 127,500    | 172,500    |

## 常见问题

### Q: 最小需要多少资金？

**答**：每格至少 5 USDT，所以：
- 10 格网格 → 至少 55 USDT
- 20 格网格 → 至少 110 USDT

建议测试资金：100-200 USDT

### Q: 网格数量设置多少合适？

**答**：
- 资金少（100 USDT）：10-15 格
- 资金中（500 USDT）：20-30 格
- 资金多（1000+ USDT）：30-50 格

网格间距建议 2-3%，太小会被手续费吃掉利润。

### Q: 价格区间怎么设？

**答**：两种方法：

**简单法**：当前价 ±15%
```
当前 100,000 → 下限 85,000，上限 115,000
```

**技术分析法**：看 K 线支撑阻力
```
支撑位（近 1 月低点）→ 下限
阻力位（近 1 月高点）→ 上限
```

### Q: 为什么我的网格不成交？

**答**：可能原因：
1. **价格没波动** - 网格需要价格波动才能成交
2. **区间太窄** - 价格直接突破区间，没有来回波动
3. **单边行情** - 只涨不跌或只跌不涨，无法形成买卖循环

### Q: 什么行情最赚钱？

**答**：**宽幅震荡**最赚钱！

| 行情类型 | 收益 | 说明 |
|---------|------|------|
| 宽幅震荡 | ⭐⭐⭐⭐⭐ | 反复穿越网格，赚最多 |
| 缓慢上涨 + 回调 | ⭐⭐⭐⭐ | 正常赚钱 |
| 横盘震荡 | ⭐⭐⭐ | 赚小钱 |
| 单边暴涨 | ⭐ | 卖飞踏空 |
| 单边暴跌 | ❌ | 满仓被套 |

### Q: 如何停止网格？

**答**：
1. Web 面板找到对应网格
2. 点击 **「停止」** 按钮
3. 系统会自动撤销挂单，卖出持仓（止损/止盈时）

### Q: 如何提取利润？

**答**：
1. 停止网格
2. 利润会自动回到 OKX 账户余额
3. 在 OKX App 提现或继续投资

## 风险提示

⚠️ **加密货币交易有风险，请谨慎投资**

| 风险 | 说明 | 应对 |
|------|------|------|
| 破网风险 | 价格跌破下限，满仓被套 | 设置止损价，及时平仓 |
| 卖飞风险 | 价格突破上限，空仓踏空 | 设置止盈价，或重新建仓 |
| 流动性风险 | 极端行情无法成交 | 选择主流币种（BTC/ETH） |
| 平台风险 | OKX 平台出现问题 | 定期提现，不要囤积 |

### 安全建议

1. **不要给 API 提现权限** - 只给读取 + 交易
2. **定期提现利润** - 不要累积太多资金在平台
3. **先用小额测试** - 建议 100 USDT 起步
4. **使用模拟盘** - OKX 有模拟交易功能
5. **不要 All in** - 只用闲钱投资
